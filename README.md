# 🍽️ Restaurant Website — Modern UI Experience

A visually rich and fully responsive restaurant website built with modern frontend technologies.
This project focuses on **smooth animations, elegant UI design, and an immersive user experience**.

---

## 🌐 Live Demo

👉 [Click Here](https://restarurant-v2.vercel.app/)

---

## 📌 Overview

This project showcases a **premium restaurant landing page** with attention to detail in design and interaction.
It is built to simulate a real-world product with sections like menu, services, and contact—combined with fluid animations and responsive layouts.

---

## ✨ Features

* 🎯 Fully responsive design (mobile, tablet, desktop)
* 🎬 Smooth animations (hover effects, transitions, scroll-based)
* 🍴 Menu & services sections with structured layout
* 📩 Contact form UI
* 🌌 Modern dark-themed aesthetic
* ⚡ Optimized performance and fast loading
* 🧩 Component-based structure (React)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Framework:** React.js (Vite)
* **Styling:** Custom CSS / SCSS
* **Animations:** CSS animations / (optional GSAP)
* **Version Control:** Git & GitHub
